## Списки

Чтобы добавить ненумерованные списки, необходимо пункиы выделить (*). Например:
* Element 1
* Element 2
* Element 3

Чтобы добавить пронумерованные списки, необходимо мункты пронумеровать. Например:
1. Элемент
2. Элемент
